¿Qué es JAMstack?

Arquitectura de desarrollo web basada en JavaScript del lado del cliente, APIs reutilizables y un Markup preconstruido

¿Qué es un stack?

Es una combinación de productos de software y lenguajes de programación usados para crear una Web o una Aplicación

JAMSTACK

  • API | Data Source, Third Party Services
  • Markup | Markdown content
